# 前期准备

## 初始参数

初始参数

```Python
"AIL-SVRG-ADMM": {
    "batch_size": 64,
    "step_multiplier": 1.00,
    "tau": 0.0,
    "varrho": 3,
    "beta_y": 0.0,
    "p_min": 0.002,
},
```



| 参数              | 当前值 | 作用                                 |
| ----------------- | ------ | ------------------------------------ |
| `batch_size`      | 64     | 控制每轮小批量梯度估计的样本数       |
| `step_multiplier` | 1.35   | 放大数据相关基步长                   |
| `tau`             | 0.10   | 控制惯性外推程度                     |
| `varrho`          | 3      | 控制前期快照刷新概率                 |
| `beta_y`          | 0.0    | 控制 \(y\) 更新的历史滞后/近端稳定项 |
| `p_min`           | 0.002  | 控制后期快照刷新概率下界             |

> ## 推荐调参顺序
>
> 不要同时调多个参数。建议：
>
> 1. 固定其余参数，确认 `step_multiplier=1.30/1.35/1.40`；
> 2. 固定最佳步长，测试 `tau=0/0.05/0.10`；
> 3. 固定最佳 `(step_multiplier, tau)`，测试 `p_min=0.001/0.002/0.003`；
> 4. 若后期 KKT 仍有平台，再测试 `batch_size=96/128`；
> 5. 仅在 primal residual 振荡时测试 `beta_y=0.01/0.05`；
> 6. 最后测试 `varrho=1/3/5`。
>
> 选择参数时以以下顺序为准：
>
> 1. KKT residual；
> 2. primal residual；
> 3. optimality gap；
> 4. time 与 IFO；
> 5. test logistic loss。
